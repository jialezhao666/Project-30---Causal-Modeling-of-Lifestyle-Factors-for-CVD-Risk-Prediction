# Project-30---Causal-Modeling-of-Lifestyle-Factors-for-CVD-Risk-Prediction
## Project 30 - Causal Modeling of Lifestyle Factors for CVD Risk Prediction code storing.
### Description:
This project uses UK Biobank lifestyle data to model how behavioral changes affect 
cardiovascular disease risk. Modifiable factors (e.g., physical activity, smoking, diet, sleep, 
alcohol) will be analyzed using causal inference and machine learning methods, including causal 
forests, to estimate individual-level effects of lifestyle changes. Logistic regression will provide 
baseline prediction with internal validation. The project will develop an interactive tool visualizing 
how lifestyle adjustments, such as quitting smoking or improving diet can alter CVD risk, 
supporting informed prevention and personalized patient engagement.
